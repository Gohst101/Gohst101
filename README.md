##### Moderation - 🔨
- Ban - /ban <user> <!reason>
- Unban - /unban {user}
- Kick - /kick {user} {!reason}
- Timeout - /timeout {user} {time} {!reason}
- Warn - /warn {user} {reason}
- Unwarn - /unwarn {user}
- Clear - /clear {amount} {!fromUser}
